# Projects

## 1. 가족같은

### 소개
`가족같은` 프로젝트는 행복한 가족 생활을 위한 소통 도구를 개발하는 것을 목표로 한 서비스입니다.  
가족 구성원들이 빠르고 쉽게 소통할 수 있도록 지원하며, 일정 관리와 공유를 통해 유대감을 형성할 수 있도록 돕습니다.

### 주요 기능
- 로그인 / 회원가입
- 게시글
- 가족 앨범
- 일정 공유
- 채팅
- 설정 및 관리

### 역할 및 성과
- 게시글 관리 기능 구현 (작성, 수정, 삭제)
- 댓글 기능 구현 (작성, 수정, 삭제)
- 사용자 간 소통을 위한 기본 커뮤니티 기능 구축

## 2. 청년돋움

### 소개
`청년돋움` 프로젝트는 청년 정책을 효율적으로 활용할 수 있도록 지원하는 서비스입니다.  
방대한 정책 정보 속에서 필요한 내용을 쉽게 찾을 수 있도록 개인 맞춤형 기능을 제공합니다.

### 주요 기능
- 로그인 / 회원가입
- 개인화된 정책 추천
- 정책 일정 및 마감일 관리 (캘린더)
- 커뮤니티 기능
- 마이페이지

### 역할 및 성과
- 게시글 기능 구현 (작성, 수정, 삭제)
- 댓글 및 대댓글 기능 구현
- 반응형 웹앱 구현 (다양한 디바이스 대응)
- 사용자 맞춤형 정보 접근성 향상

## 3. Wego

### 소개
`Wego`는 즉흥 여행을 돕는 서비스로, 여행 계획의 부담을 줄이고 새로운 경험을 제공하는 것을 목표로 합니다.  
랜덤 여행지 추천과 미션 기반 활동을 통해 사용자 참여형 여행 경험을 제공합니다.

GitHub: https://github.com/UMC-WEGO/WEGO_FE_React

### 주요 기능
- 로그인 / 회원가입
- 여행지 추천
- 일정 관리
- 게시판
- 스크랩 관리
- 마이페이지 및 설정

### 역할 및 성과
- 게시판 기능을 **즉흥 여행 후기 공유 플랫폼**으로 고도화
- 사용자 간 미션 제안 및 추천 시스템 구현
- 커뮤니티 참여율 증가에 기여
- 여행 지역 선택 기반 데이터 처리 로직 구현
- 지역별 콘텐츠 탐색 효율 향상

## 4. 고유한학습터

### 소개
`고유한학습터`는 초등 임용고시 준비를 위한 학습 서비스입니다.  
반복 학습과 오답 중심 복습을 통해 효율적인 암기 학습을 지원합니다.

- 학습 모드
- 복습 모드
- 커뮤니티
- 결제
- 마이페이지 및 설정

### 특징
- 반복 학습 기반 암기 시스템 제공
- 오답 중심 학습 경험 설계
- 학습자 간 커뮤니티 지원

## 5. Devine

### 소개
`Devine`은 프로젝트와 개발자를 연결하는 매칭 플랫폼입니다.  
개발자의 역량을 분석하고, 프로젝트에 적합한 인재를 추천하는 것을 목표로 합니다.

### 주요 기능
- 프로젝트 / 개발자 매칭
- 기술 스택 기반 필터링
- 지원 및 상태 관리 시스템
- GitHub 코드 분석 리포트 생성

### 역할 및 성과
- 프론트엔드 팀장으로 참여
- 역할 분배 및 일정 관리
- 사용자 흐름 기반 UI 설계 및 구현
- GitHub 코드 분석 Report 생성 기능 기획 및 구현
- 프로젝트/개발자 추천 알고리즘 설계 및 구현
